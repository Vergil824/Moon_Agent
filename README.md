# Moon

本仓库包含一个 Next.js 应用，位于 `moon-agent/`。

## 环境要求

- Node.js: `>=18.18.0`
- npm: `>=10`（本项目声明 `npm@10.5.2`）

## 安装依赖

在仓库根目录执行：

```bash
cd moon-agent
npm install
```

## 启动开发环境

```bash
cd moon-agent
npm run dev
```

启动后请打开终端输出的地址（Next.js 默认是 `http://localhost:3000`）。

## 构建与生产启动

```bash
cd moon-agent
npm run build
npm run start
```

## 运行测试

```bash
cd moon-agent
npm test
```
